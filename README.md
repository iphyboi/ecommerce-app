       E-commerce App

A modern e-commerce web application with product listing, authentication, and cart functionality built using Next.js.

       Live Demo

https://ecommerce-app-two-pi.vercel.app

         Tech Stack

Next.js

MongoDB

Mongoose

Tailwind CSS

REST API



            Features

- User authentication (login/registration)
- 
- Product listing and display
- 
- Add to cart functionality
- 
- API for managing products
- 
- Responsive design

             Project Structure

  - `/app` - Frontend pages
  - 
  - `/api` - Backend API routes
  - 
  - `/models` - Database schemas
  - 
  - `/components` - Reusable UI components
